Camera_Photo文件夹保存前置摄像头所拍照片
Upload_Detected文件夹保存本地上传图片
Face_Detected文件夹保存人脸图片

TanBinKai_16081119_JAVA_fat.jar为导出jar包
TanBinKai_16081119_JAVA.exe为可执行程序

1.本地上传图片
点击主页面“上传图片按钮”进行上传；
点击“本地上传图片的人脸识别+随机森林颜值打分”
进行颜值打分

2.摄像头拍照打分
点击左侧摄像头“拍照按钮”（需有红框框住人脸，如果没有脸部信息将无法识别）
点击“前置摄像头拍照+人脸识别+颜值打分”进行打分

每次程序关闭后都应清空Camera_Photo、Upload_Detected、Face_Detected文件夹内图片，以便下次使用。
